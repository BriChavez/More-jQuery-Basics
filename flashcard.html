jQuery Basics: Flashcards for studying

  • Operators

    • - Performs some operation on single or multiple operands (data value) and produces a result.

  • Variables

    • - Are containers for storing data (storing data values).

  • Variable naming conventions

    • - let, var, and const

  • Functions

    • - A function is a bundle of code that performs a set of procedures.

  • Methods

    • - A method is a type of function. It is an action run on a piece of data; you can think of it as a message you send to a piece of data, and the result is the response.
